FleetDB is a record-oriented database optimized for agile development.

FleetDB offers a schema-free data model that is expressive and flexible. Declarative queries, automatically maintained indexes, and an optimizing query planner make it easy to manipulate and retrieve data. Keeping all data in RAM as persistent data structures enables multi-document transactions and gives excellent performance. An append-only log provides durability in a single file that can be transfered and analyzed using standard Unix tools. FleetDB is accessible via a simple JSON-based protocol that is implemented by clients in several programming languages.
